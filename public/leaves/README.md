# Leaf images for the light-mode water scene

Drop real leaf photos here and `PondCanvas` will use them automatically (no code
change needed). If this folder is empty, the scene falls back to drawn leaves.

## Naming
`leaf-01.png`, `leaf-02.png`, … up to `leaf-20.png` (any count; gaps are fine).

## What each image should be
- **Transparent background (PNG with alpha).** This is essential — the leaf must be
  cut out, no white/colored box behind it.
- **Top-down / flat view** of a single leaf, as if lying on water seen from above.
- Leaf roughly fills the frame with a little padding around it.
- ~256–512 px on the long side.
- A mix of types and colors works best: autumn maple (red / orange / yellow),
  aspen or birch (yellow, yellow-green), a couple of plain green leaves, and
  optionally 1–2 lily pads.

Once added, hard-refresh the page.
